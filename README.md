# x-note cli

CLI app running on Node.js.

Note
  - id: string
  - text: string
  - done: boolean
  - status: enum active, deleted
  - updated: date

CLI
  - show all/summary
  - filter all/active/archived
  - sort text/done/status/updated
  - add 'note-text'
  - done/delete 'note-index'
  - purge
